## Ultrasonic radar using Arduino Uno R3

<p><b>
Low cost ultrasonic radar design, using arduino and the Tinkercad platform.
</b></p>

## Components used in the project

|Amount  |      Component      |  
|----------|:-------------:|------:|
| 1 |  Arduino Uno R3 | 
| 1 |    LCD 16x2   |
| 1 | 10 kΩ Potentiometer| 
| 1 |  220 Ω Resistor | 
| 1 |    Micro servo   |
| 1 | Ultrasonic distance sensor |
